◆ tick()

void LoadBalancer::tick ( int  clock_cycle)
inline

Advances the simulation by one clock cycle, processing requests and adjusting servers as needed.

Parameters
clock_cycleThe current clock cycle.

The function checks each server for availability and assigns requests from the queue. It also dynamically adds or removes servers based on the queue size and current server count, with a cooldown period of 10 cycles to prevent rapid fluctuations.
